## 8. Tastaturkürzel

- `⌘ A` / `Strg A` — Alle Pylone auswählen
- `⌘ C` / `Strg C` — Auswahl kopieren
- `⌘ V` / `Strg V` — Einfügen (mit 1 m Versatz)
- `⌘ Z` / `Strg Z` — Rückgängig
- `⌘ ⇧ Z` / `Strg Y` — Wiederherstellen
- `Entf` / `⌫` — Auswahl löschen (Pylon, Pfeil oder Maßlinie)
- `Esc` — Zurück in den Auswahl-Modus
