## 4. Breite messen

Mit dem Werkzeug **Breite messen** klickst du nacheinander zwei Pylone an. Der Editor berechnet daraus die **lichte Breite** (Abstand abzüglich der Pylonengröße), trägt sie in das Feld „Lichte Breite" ein und zeichnet eine Maßlinie auf der Fläche ein.

Eine ausgewählte Maßlinie zeigt ihre Länge in der Fußzeile unter dem Canvas — von dort kannst du sie per Klick als „Lichte Breite" übernehmen oder wieder löschen. Liegt die gemessene Breite unter der Mindest-Aufgabenbreite, wird das mit einer Warnung markiert.
