## 5. Versionen

Für angemeldete Nutzer (Pro-Tarif) kannst du zu jeder Strecke manuelle **Snapshots** anlegen — eine automatische Versionshistorie gibt es nicht, du entscheidest selbst, wann ein Stand gesichert wird.

Im Dashboard siehst du zu jeder Strecke die Liste ihrer Versionen. Dort kannst du:

- eine Version **read-only ansehen**, ohne die aktuelle Strecke zu verändern
- die aktuelle Strecke auf eine Version **zurücksetzen** (Sicherheitsabfrage, da der aktuelle Stand dabei überschrieben wird)
- eine Version als **neue, eigenständige Strecke** speichern, statt die aktuelle zu überschreiben
- eine Version löschen
