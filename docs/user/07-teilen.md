## 7. Teilen

Als angemeldeter Nutzer (Pro-Tarif) kannst du für eine Strecke einen **Share-Link** erzeugen. Wer den Link öffnet, sieht die Strecke read-only als Grafik — ganz ohne eigene Anmeldung und ohne Kartenhintergrund.

Pro Strecke ist immer nur **ein Link aktiv**: Erzeugst du einen neuen, wird der vorherige ungültig. So behältst du die Kontrolle darüber, wer aktuell Zugriff hat.
