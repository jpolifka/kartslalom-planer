## 4. Speichern

**Ohne Anmeldung (Gast):** Deine Strecke wird automatisch in diesem Browser gespeichert (Anzeige „Gespeichert" in der Werkzeugleiste) und beim nächsten Besuch wiederhergestellt. Es wird nichts an einen Server übertragen. Über „Neu" setzt du die Strecke zurück und löschst den gespeicherten Stand (mit Sicherheitsabfrage).

**Mit Anmeldung:** Deine Strecken werden zusätzlich in der Cloud gespeichert, im Dashboard verwaltet und sind von jedem Gerät aus erreichbar. Wie viele Strecken du gleichzeitig anlegen kannst, hängt von deinem Tarif ab (Free: 3, Pro: 50, Team: unbegrenzt).
