## 4. Speichern

**Ohne Anmeldung (Gast):** Deine Streckendaten werden ausschließlich in diesem Browser gespeichert und beim nächsten Besuch wiederhergestellt. Der Speicherstatus wird dabei in der Werkzeugleiste angezeigt. Es erfolgt keine Speicherung auf dem Server. Über „Neu" setzt du die Strecke zurück und löschst den gespeicherten Stand (mit Sicherheitsabfrage).

**Mit Anmeldung:** Deine Strecken werden zusätzlich in deinem Benutzerkonto gespeichert, im Dashboard verwaltet und sind von jedem Gerät aus erreichbar. Wie viele Strecken du gleichzeitig anlegen kannst, hängt von deinem Tarif ab (Free: 3, Pro: 50, Team: unbegrenzt).
